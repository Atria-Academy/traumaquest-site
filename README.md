# TraumaQuest — Landing Page

Landing page da comunidade TraumaQuest.

## Deploy na Vercel

1. Suba este repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e faça login com sua conta GitHub
3. Clique em **"Add New Project"**
4. Selecione o repositório `traumaquest-site`
5. Em **Framework Preset**, selecione **"Other"**
6. Em **Output Directory**, confirme que está `public`
7. Clique em **Deploy**

Pronto. A Vercel vai gerar um link automático (ex: `traumaquest-site.vercel.app`).

## Domínio customizado

Após o deploy, em **Settings > Domains** você pode adicionar seu domínio próprio.
